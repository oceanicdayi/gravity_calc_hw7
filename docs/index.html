Gravity Station Calculation

Given Data:

  • Latitude: 48.1195¬∞ N
  • Longitude: 12.1878¬∞ E
  • Elevation: 487.9 m
  • Observed gravity: 980,717.39 mGal

Results

  • Theoretical gravity (gtheo) = 980,619.38 mGal
  • Free-air correction (FAC) = +147.97 mGal
  • Bouguer correction (BC) = -111.54 mGal
  • Free-air anomaly = +245.98 mGal
  • Bouguer anomaly = +134.44 mGal

Case (b): Station over the ocean (depth = 487.9 m)

  • Free-air anomaly = +245.98 mGal (same correction)
  • Bouguer correction (for water) = -16.23 mGal
  • Bouguer anomaly = +229.75 mGal

Python Code

import math

# Given data
lat = 48.1195  # degrees
elev = 487.9   # m
obs_g = 980717.39  # mGal

# 1. Theoretical gravity (IGF80)
g_theo = 978032.67715 * (1 + 0.0053024 * math.sin(math.radians(lat))**2 - 0.0000058 * math.sin(2 * math.radians(lat))**2)

# 2. Free-air correction
FAC = 0.3086 * elev  # mGal/m

# 3. Bouguer correction (rock)
rho_rock = 2.67  # g/cm^3
BC = 0.04193 * rho_rock * elev

# 4. Anomalies
FA_anom = obs_g - g_theo + FAC
BA_anom = obs_g - g_theo + FAC - BC

print(f"Theoretical gravity: {g_theo:.2f} mGal")
print(f"Free-air correction: {FAC:.2f} mGal")
print(f"Bouguer correction: {BC:.2f} mGal")
print(f"Free-air anomaly: {FA_anom:.2f} mGal")
print(f"Bouguer anomaly: {BA_anom:.2f} mGal")

# 5. For ocean station (water density)
rho_water = 1.03
BC_water = 0.04193 * rho_water * elev
BA_anom_water = obs_g - g_theo + FAC - BC_water

print(f"\nFor ocean case:")
print(f"Bouguer correction (water): {BC_water:.2f} mGal")
print(f"Bouguer anomaly (water): {BA_anom_water:.2f} mGal")
